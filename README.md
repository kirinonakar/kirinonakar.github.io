# GitHub Pages Test Site

이 저장소는 GitHub Pages를 위한 테스트 웹 페이지입니다. 프리미엄 디자인과 반응형 레이아웃이 적용되어 있습니다.

## 🚀 배포 방법

1. 이 저장소를 GitHub에 업로드합니다.
2. 저장소의 **Settings** 탭으로 이동합니다.
3. 왼쪽 메뉴에서 **Pages**를 클릭합니다.
4. **Build and deployment** 섹션의 **Source**가 `Deploy from a branch`로 되어 있는지 확인합니다.
5. **Branch**를 `main` (또는 해당 브랜치)와 `/ (root)`로 설정하고 **Save**를 누릅니다.
6. 잠시 후 배포된 URL이 상단에 나타납니다!

## ✨ 주요 기능

- **모던 디자인**: 유리질감(Glassmorphism)과 다크 모드 테마.
- **Featured Repositories**: `kirinonakar`님의 주요 공개 저장소 실시간 소개 섹션.
- **반응형**: 모바일과 데스크톱 모두 최적화.
- **인터랙티브**: 마우스 이동에 따른 배경 그래디언트 애니메이션.
- **SEO 최적화**: 기본적인 메타 태그 포함.
